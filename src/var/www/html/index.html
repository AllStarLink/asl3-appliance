This is your new ASL3 Node

AllStarLink is proud to announce ASL3, the next generation of AllStar repeater and hotspot software. ASL3 has been redesigned to run on Asterisk LTS (Long Term Support), the latest operating systems, and modern hardware. The update from Asterisk version 1.4 to version 20 implements over 15 years of bug fixes, security improvements and enhancements.

Of course AllStar benefits from running on Asterisk 20. But we didn't stop there. Our team has worked hard over the last two plus years to bring you many AllStar improvements. Our code is available to all. You may read all about the many new features, improvements and how to install on our new ASL3 Manual.

AllStarLink Community is both an email server and forum software. You can use it in either or both modes.

The Community is the best place to ask questions that you don't find answered elsewhere and is a great place to engage with your fellow AllStar users.

Allmon3 Node Monitoring & Management

Allmon is the standard web-based montitoring and management for the AllStarLink application. Allmon3 is the next generation of the venerable Allmon2 that is rewritten for performance, scalability, and responsiveness.

Note that while the default installation of Allmon3 has already been done, you will still need to set the initial password for the Allmon3 default user. You can use the terminal available in Cockpit to login to the operating system to perform this step.

Please see the Configure Allmon section of the Allmon3 Git repository for assitance in performing this step.

Cockpit Console - Node OS Management

Cockpit console is a web-based graphical interface to your node. By using it, you can perform system administration tasks, such as inspecting and controlling services, managing storage, configuring networks and firewalls, analyzing network issues, and inspecting logs. Of course, it's also very handy for administering Asterisk on your AllStarLink Pi Appliance!

Note that you will use the operating system credentials created during the installation process of your node.

Please see the ASL3 User Guide for further information on the use of Cockpit.
